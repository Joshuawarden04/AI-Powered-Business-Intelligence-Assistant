# AI-Powered-Business-Intelligence-Assistant
AI-powered business intelligence assistant using Tableau MCP and Claude for natural language analytics and automated insights.

## Overview
This project integrates Tableau dashboards with Claude MCP to provide AI-assisted business analytics through natural language queries.

## Features
- Natural language interaction with business data
- Automated insight generation
- Executive dashboard reporting
- KPI tracking and visualization
- Business performance analysis

## Technologies Used
- Tableau
- Python
- Data Analytics
- Business Intelligence
